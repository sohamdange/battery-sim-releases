Capabilities

Everything you need to simulate a battery pack

From cell characterisation to pack-level drive-cycle analysis — all in one desktop application.

Cell Library
Import and manage a reusable catalogue of cell models. Define capacity, OCV curves, voltage bounds, and RC network parameters. Upload tabular RC data per temperature via CSV — cylindrical, prismatic, pouch, or coin form factors.
Pack Builder
Configure multi-cell packs with arbitrary series and parallel topology. Automatic busbar resistance calculation, geometry-aware cell classification, and full electrical property derivation from your selected cell model.
Drive Cycle Simulation
Import standard (WLTP, UDDS) or custom drive cycles. Compose multi-step simulation schedules with subcycle extraction. Map drive cycles to calendar days across a full year for long-horizon mission profiles.
RC Equivalent-Circuit Solver
KCL-based solver models every cell in the pack individually — no lumped approximations. Compiled with Cython for speed. Handles arbitrary topology including unbalanced parallel branches and mixed cell types within the same pack.
Results & Analysis
Results written to HDF5: full voltage, current, state-of-charge, and temperature traces at both pack and per-cell resolution. Inspect plots directly in the app — pack level, cell level, spatial, and overlapping views with unit controls.
Offline & Local-First
No cloud, no internet required to run simulations. All cell models, pack configurations, drive cycles, and results are stored locally. Single installer — Python, runtimes, and all libraries are bundled. Nothing else to install.
