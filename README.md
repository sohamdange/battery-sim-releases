# Battery Simulation Toolkit — Releases

Installer releases for **Battery Simulation Toolkit**, a desktop application for battery pack simulation using equivalent-circuit models.

**Product website:** https://sohamdange.github.io/battery-sim-releases

## Download

Download the latest Windows installer from the [Releases](https://github.com/sohamdange/battery-sim-releases/releases/latest) page.

## System Requirements

- Windows 10 or Windows 11 (64-bit)
- ~500 MB disk space
- 4 GB RAM minimum (8 GB recommended for large packs)
- No additional runtime or Python installation required
